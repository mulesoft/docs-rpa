// == What's New
include::partial$release-notes/rn-headings.adoc[tag=whatsNew,leveloffset=+1]

* The Tomcat dependency is upgraded to version 9.0.87 to address reported security vulnerabilities.
* The Gecko Driver for Firefox is now updated. 
* The setup now inclues the latest version of the Chrome and Edge drivers.
