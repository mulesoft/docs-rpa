== September 29, 2022

// == What's New
include::partial$release-notes/rn-headings.adoc[tag=whatsNew,leveloffset=+2]

== August 1, 2022

// == What's New
include::partial$release-notes/rn-headings.adoc[tag=whatsNew,leveloffset=+2]

MuleSoft announces the release of RPA Manager, the cloud-based control plane that enables you to:

* Manage user access and permissions.
* Evaluate the viability and financial benefits of implementing an automation.
* Design and document RPA processes.
* Test and deploy automations.
* Manage and monitor deployed automations.
