[[september-29-2022]]
== September 29, 2022

// == What's New
include::partial$release-notes/rn-headings.adoc[tag=whatsNew,leveloffset=+2]

* The *Download Applications* section on the landing page is now collapsible, which enables you to see the *Need More Help* section without scrolling.
* The videos from the *Learning Center* now show their duration.
* The usage summary now contains a month selector. Thus, the user no longer needs to specify a time range with start and end dates.
* The image of an application is now limited to a maximum size of 5 MB.
* A user's profile image is now limited to a maximum size of 5 MB.

== August 1, 2022

// == What's New
include::partial$release-notes/rn-headings.adoc[tag=whatsNew,leveloffset=+2]

MuleSoft announces the release of RPA Manager, the cloud-based control plane that enables you to:

* Manage user access and permissions.
* Evaluate the viability and financial benefits of implementing an automation.
* Design and document RPA processes.
* Test and deploy automations.
* Manage and monitor deployed automations.
