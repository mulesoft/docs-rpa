== Loop

A _Loop_ allows you to repeat the same _Transaction_ and _Action Steps_
two or more times.

=== Properties

* *At end of loop* Sets how the _Loop_ should be exited after all
iterations have been executed. Available options are _Exit with error_
or _Exit without error_.
* [#AS_Loop_P_IncrementValue .anchor]####**Increment Value** Value the
_Loop_ counter increments by on each iteration
* [#AS_Loop_P_IterationCount .anchor]####**Iteration Count** Sets how
often the _Loop_ will be executed. Default value is 2.
* [#AS_Loop_P_StartValue .anchor]####**Start Value** Value the _Loop_
counter starts with
